## 🧠 AI Models & APIs

| Tool | URL | Notes |
|---|---|---|
| Anthropic Claude API | [anthropic.com](https://www.anthropic.com/) | Claude Haiku recommended for cost efficiency; Sonnet as fallback for complex queries |
| Claude Haiku | [docs.anthropic.com](https://docs.anthropic.com/) | Fast, low-cost model — good starting point for civic Q&A |
| Claude Sonnet | [docs.anthropic.com](https://docs.anthropic.com/) | More capable; use when Haiku falls short |
| NanoGPT | [nano-gpt.com](https://nano-gpt.com/) | Subscription-based access to multiple models at lower cost |
| Meta Llama (open source) | [llama.meta.com](https://llama.meta.com/) | Free to run locally; potential cost-free alternative once validated |
| Mistral (open source) | [mistral.ai](https://mistral.ai/) | Another strong open-source option for local deployment |
